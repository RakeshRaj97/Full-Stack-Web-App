Enhancement 1: Image Transition

The code required to implement this feature are,

  • hover
  • transition
  • opacity
  • visibility
  • width

This feature is used for transistion when the mouse is hovered over the image. This feature is applied to topic.html page. Whenever the mouse is hovered, we can see the image zoomed from 45% to 65% of its width. We can also see that the text is hidden and visible only on hover. This feature can be accessed using this link.

Enhancement 2: Image slideshow

The code required to implement this feature are,

  • animation
  • animation-timing-function
  • animation-play-state
  • keyframes(Rule)
  • left
  • opacity

This feature introduces image sliding in topic.html page using animation. The width of each image is made to 100% of the page width. The animation name is called slide and used on the element figure. The keyframe rules defines the slide frame by frame. The element figure is pushed gradually from 0% to -300% (1st slide to last slide) and finally returns back to the initial slide at the end. The opacity is modified to give a transistion effect. The animation-timing-function is used for a constant speed from start to end.

This feature can be accessed using this link.
