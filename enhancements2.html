Detailed description of enhancements used for Javascript

Enhancement 1: Implement a Timer

The code required to implement this feature are,

  • setTimeout
  • clearTimeout
  • alert
  • window.location.assign

This feature uses a timer so that the user has only a limited time to complete the quiz. The function "startTimer()" in "enhancements.js" is used to implement this feature.
A timer of 60 seconds is set for quiz completion, failing which the form is redirected to "result.html" page and the attempt count is stored in localStorage. View the timer using this link.

Enhancement 2: Confirmation box to display instructions before quiz start.

The code required to implement this feature are,

  • confirm
  • window.location.assign

This feature introduces a confirmation box to the user before they begin the quiz.
The confirmation box holds the instructions regarding to the time limit and the scoring criteria. If the user selects "OK" then a message "All the best" is displayed which can be viewed using this link and the user can start the quiz. If a user selects "Cancel", he is redirected to the "index.php" page.
