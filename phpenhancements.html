Enhancement 1: Login for Quiz Supervisors

The code required to implement this feature are,

  • @mysqli_connect
  • mysqli_query
  • mysqli_affected_rows
  • SQL Query

This features introduces a creation of new table in database called "manage" which stores the username and password of quiz supervisors. The access to manage the records is possible only by login using the correct supervisor credentials.

Enhancement 2: Sort the order to display the quiz attempt records

The code required to implement this feature are,

  • @mysqli_connect
  • mysqli_fetch_assoc
  • mysqli_affected_rows
  • SQL Query

This feature allows the supervisor to sort and list all records of the attempts table. The supervisor is provided the various options to sort such as Score, Student ID, First Name, Last Name, Date and Time, Number of Attempts and No Sort.

This feature is used in the "supervisor.php" page which is accessible only after successful login of "manage.php".
